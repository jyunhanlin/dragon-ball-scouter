# Dragon Ball Scouter

瀏覽器玩具:相機畫面 → 臉部偵測 → 「戰鬥力」計算 → 掃描器 HUD。使用者對著鏡頭大吼就能變身超級賽亞人。

## Language

### 變身與狀態

**變身 (Transformed)**:
從 SSJ 狀態開始、延續整個 overload 的期間;判準是「變身時間戳存在」,不是單一 FSM phase。頭髮特效只在變身期間存在。
_Avoid_: SSJ 中(只指 FSM 的 ssj phase,不含 overload)

### 頭髮

**髮束 (Spike)**:
賽亞人頭髮的最小造型單位——一撮尖刺狀的頭髮。整個髮型由髮束的分佈、長度、彎曲定義。
_Avoid_: 圓錐、cone(那是舊實作的形狀,不是領域概念)

**造型基準 (Silhouette Reference)**:
「像不像賽亞人」的仲裁標準:悟空 SSJ1 的正面剪影。所有造型 review 以此為準,不憑感覺。

**模擬三維度**:
「高度模擬」拆成三個互相獨立的維度——造型神似(silhouette)、物理擬真(motion)、貼合真人(fit)。談「擬真度」時必須先指明是哪一個維度。

**正統賽璐璐 (Full Cel)**:
含暗部色階與描邊的動畫上色。與「亮域賽璐璐」(把色階壓進只能加亮的色域、犧牲暗部)相對。本專案的金髮走正統賽璐璐。
_Avoid_: toon shading(太泛,未指明有無暗部)

**頭皮圓頂 (Scalp Dome)**:
從臉部 landmark 擬合出的半橢球面,髮束根部沿它分佈——讓頭在正面 ±40° 內轉動時髮型保有體積感。

**豎起演出 (Rise)**:
變身瞬間髮束從垂軟到彈簧式豎立的演出;是「變身」體驗的高峰,取代單純的縮放漸入。

**慣性甩動 (Inertia Sway)**:
頭部運動時髮束滯後、回彈、阻尼收斂的動態;「頭髮是活的」的最低門檻。三種 M2 動態之一。

**氣場上飄 (Aura Updraft)**:
靜止時髮束像火焰般微微搖曳上飄的動態;SSJ 的標誌性能量感。三種 M2 動態之一。

**吼叫連動 (Yell Reactivity)**:
effort 訊號驅動髮束豎立度、飄動強度與金光的互動回饋;「用力=變強」。三種 M2 動態之一。

**染金 (Gold Tint)**:
貼合真人的機制:用 hair mask 將使用者真實頭髮調色成金色(保留真實髮絲質感與髮際線),髮束只負責補出尖刺形狀。
_Avoid_: 蓋髮、換髮(那是被否決的「不透明覆蓋」方案)
